# Product Metrics Dashboard

## Описание проекта
Проект демонстрирует анализ продуктовых метрик интернет-магазина:
- DAU, WAU, MAU
- Retention
- Топ категорий по выручке

## Данные
- users.csv — пользователи и дата регистрации
- orders.csv — заказы и сумма
- order_items.csv — товары и категории

## Метрики
- Daily Active Users (DAU)
- Revenue by category
- Retention (1 день после регистрации)

## Инструменты
- Python (pandas, matplotlib, seaborn)
- SQL
- Jupyter Notebook

## Вывод
Проект показывает ключевые продуктовые метрики, их визуализацию и анализ поведения пользователей.
